# case1: 中等规模角色协作测试
你需要模拟20个不同角色协同编辑Wikipedia词条"人工智能伦理"的完整过程。

**第一阶段：角色生成**
生成20个具有不同专业背景的角色，包括：
- AI研究员、伦理学家、法律专家、社会学家、心理学家
- 科技公司高管、政策制定者、记者、教育工作者
- 不同文化背景和价值观倾向的代表

**第二阶段：协同编辑**
每个角色根据自己的专业背景对"人工智能伦理"词条提出编辑建议：
- 技术专家关注算法公平性、透明度
- 伦理学家关注道德原则、价值冲突
- 法律专家关注监管框架、责任归属
- 社会学家关注社会影响、不平等问题

识别主要分歧点，如：
- 监管严格程度的分歧
- 技术发展与伦理约束的平衡
- 不同文化价值观的冲突

**第三阶段：投票与共识**
实现"多数决投票"机制：
1. 对每个分歧点进行投票
2. 考虑专业权威性的权重分配
3. 处理投票结果，形成初步共识
4. 对剩余分歧进行调解和妥协

**第四阶段：最终确认**
通过类似区块链的验证机制确认最终版本：
1. 验证投票过程的公正性
2. 确认共识的有效性
3. 生成最终的词条内容
4. 处理可能的异议和分叉

要求：体现不同角色的观点差异，展示复杂的协商过程，最终达成可接受的共识。

# case2: 大规模角色协作测试
模拟50个角色协同编辑Wikipedia词条"量子计算"，使用"权威加权投票"共识机制。

**角色扩展要求：**
- 包含更多细分专业领域的专家
- 不同国家和地区的研究者
- 产业界、学术界、政府部门的代表
- 不同技术路线的支持者（超导、离子阱、光量子等）

**协同编辑复杂化：**
- 技术细节的争议（量子优势、纠错方案）
- 应用前景的分歧（时间预期、应用领域）
- 安全性考虑（量子密码、国家安全）
- 投资和发展策略的不同观点

**权威加权投票机制：**
1. 根据专业背景和贡献度分配权重
2. 学术声誉、发表论文、工程经验等因素
3. 不同类型问题采用不同的权重策略
4. 处理权威性争议和权重分配公平性

**高级共识算法：**
- 多轮投票和迭代改进
- 专家小组仲裁机制
- 分层决策（技术细节vs总体方向）
- 异议记录和少数意见保护

要求：展示大规模协作的复杂性，权威性判断的困难，以及如何在专业分歧中达成平衡。

# case3: 超大规模角色与区块链共识
模拟100个角色协同编辑Wikipedia词条"元宇宙技术"，使用"拜占庭容错共识"算法。

**超大规模角色管理：**
- 技术专家：VR/AR、区块链、AI、网络、硬件
- 内容创作者：游戏设计师、艺术家、作家
- 商业代表：平台公司、硬件厂商、内容提供商
- 社会角色：教育工作者、心理学家、社会学家
- 监管方：政府官员、法律专家、标准制定者
- 用户代表：不同年龄段、文化背景的用户

**复杂分歧处理：**
- 技术标准之争（开放vs封闭生态）
- 隐私和数据安全的不同标准
- 虚拟经济模式的分歧
- 社会影响评估的差异
- 监管框架的国际差异

**拜占庭容错共识实现：**
1. 假设存在恶意或错误的参与者
2. 实现3f+1容错机制（f为恶意节点数）
3. 多轮消息传递和验证
4. 处理网络分区和通信延迟
5. 确保最终一致性和安全性

**高级协调机制：**
- 分组讨论和代表制
- 专题工作组和交叉验证
- 冲突调解和仲裁程序
- 版本控制和变更追踪
- 共识历史记录和审计

**最终输出要求：**
1. 详细的角色分工和观点映射
2. 完整的分歧识别和分类
3. 具体的共识算法执行过程
4. 最终词条的完整内容
5. 整个过程的决策审计日志

要求：展示超大规模协作的组织能力，复杂共识算法的理解和实现，以及在高度分歧环境下达成共识的策略。

# case4: 动态角色与实时共识
模拟动态环境下的角色协作：角色可以中途加入或退出，需要实时调整共识机制。

**动态场景设计：**
- 初始30个角色开始编辑"自动驾驶技术"词条
- 过程中有新专家加入（突发技术进展）
- 部分角色因利益冲突退出
- 外部事件影响（事故、政策变化）

**实时共识调整：**
- 动态权重重新分配
- 已有共识的重新验证
- 新加入者的快速融入机制
- 退出者影响的消除处理

**容错和恢复机制：**
- 处理角色行为异常
- 网络攻击和恶意操作防护
- 系统故障的恢复策略
- 数据一致性保证

要求：展示动态环境下的适应能力和实时共识调整能力。

# case5: 跨文化价值观冲突解决
模拟来自不同文化背景的角色在编辑"全球化与本土化"词条时的价值观冲突和解决过程。

**文化多样性设计：**
- 东西方价值观差异
- 发达国家vs发展中国家视角
- 不同宗教和哲学传统
- 个人主义vs集体主义倾向

**价值观冲突处理：**
- 识别深层价值观分歧
- 寻找共同价值基础
- 文化敏感性考虑
- 包容性表达方式

**跨文化共识机制：**
- 文化代表性确保
- 翻译和理解偏差处理
- 文化中立的表达方式
- 多元化观点的平衡展示

要求：展示跨文化理解能力和价值观冲突的调和能力。
